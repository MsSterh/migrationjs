# migration.js
## Description

Library for migration creating.
It is a function with 3 params:

- 'version' is a current version of data
- 'data'
- 'migrations' is an array of migrations functions

The result of library is object:

```sh
{
  'version': version,
  'data': newData
}
```

## Usage

```sh
npm install
npm start
```

## Tests

```sh
npm test
```
